# Proyecto-front-back
Proyecto en el cual se integra el back-end y front-end con Javascript, base datos MongoDb. Utilizando un sistema de login y registro a traves de tokens (jwt) para poder acceder a CRUD de tareas. 
Para iniciar:

//En terminal 
- FRONT
  
- cd client
- npm i
  
- npm run dev


-  SERVER
-  npm run dev
